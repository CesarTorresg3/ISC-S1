# ISC-S1
Glosario
1.-Actuadores:
Son dispositivos mecánicos que transforman energía en un movimiento mecánico, que permiten controlar procesos en diversas aplicaciones.

2.-Apis:
Interfaz de programación de aplicaciones, un conjunto de protocolos y definiciones que se utiliza para el desarrollo de aplicaciones que permite la comunicación entre aplicaciones de software siguiendo un conjunto de reglas.

3.-Backups:
Parte importante de cualquier plan de seguridad de datos, es el proceso de hacer copias de datos para poder restaurar los datos originales en caso de perdida de los mismos.

4.-BIOS:
Se encarga de inicializar y verificar diferentes componentes del sistema para afirmar y permitir el arranque de la computadora, es el cerebro que pone en marcha todos los componentes del ordenador par que trabajen coordinadamente.

5.-CI/CD:
Integración continua/ despliegue continuo, es una práctica de desarrollo de software habilitada por la automatización, reducen errores y defectos en el código.

6.-Devops:
Es un conjunto de prácticas que combinan el desarrollo de software y las operaciones de TI. Tiene como objetivo agilizar el desarrollo y permitir la entrega continua de software de alta calidad.

7.-Docker:
Es una plataforma de software que le permite crear, probar e implementar aplicaciones rápidamente, puede implementar y ajustar la escala de aplicaciones rápidamente en cualquier entorno con la certeza de saber que su código se ejecutará.

8.-Sistema Embebido:
Es un sistema diseñado para ejecutar tareas específicas dentro de un dispositivo más grande. Está configurado para desempeñar una función dedicada y no se utiliza como una computadora general.

9.-Firmware:
Actúa como el puente esencial entre el hardware y las aplicaciones de software y representa una de las bases más importantes de la informática además de la electrónica moderna. Se encuentra en prácticamente todos los dispositivos que utilizamos: desde servidores y equipos de red hasta teléfonos móviles.

10.-Git:
Es una plataforma eficiente, flexible y capaz de gestionar proyectos de cualquier tamaño, permite trabajar en un proyecto de forma simultánea sin interferir en los procesos.

11.-Hardware:
Son los componentes físicos y tangibles de una computadora o sistema electrónico, a diferencia del software, que existe como código o datos, el hardware es el equipo que se puede ver, tocar e incluso oír. Es la base que permite que funcionen todas tus aplicaciones, programas y sistemas favoritos.

12.-IoT:
Es la red colectiva de dispositivos conectados y a la tecnología que facilita la comunicación entre los dispositivos y la nube, así como entre los propios dispositivos.

13.-LaaS:
Es un modelo de servicio que permite a los clientes acceder a software y licencias de manera flexible y escalable.

14.-Ley de moore:
Indica la cantidad de transistores que podemos colocar en un área determinada y que, por regla general, es obvio que podría tener un impacto en otras cuestiones, pero no necesariamente se refiere a ellas.

15.-Phishing:
Es una de las formas más extendidas y peligrosas de ciberdelincuencia en la actualidad, es un intento fraudulento de obtener información sensible haciéndose pasar por una entidad de confianza en las comunicaciones electrónicas.

16.-Software:
Es un conjunto de reglas o programas que dan instrucciones a un ordenador para que realice tareas específicas, puede utilizarse para gestionar datos, automatizar procesos y crear aplicaciones o productos informáticos.
